# Language Translation Tool

## Description

This project is a simple Language Translation Tool developed using Python.
It allows users to translate text from one language to another.

## Features

- Translate text between languages
- Easy to use interface
- Fast translation
- Built using Python

## Technologies Used

- Python
- deep-translator library

## How to Run

1. Install Python

2. Install package:

pip install deep-translator

3. Run program:

python translator.py

## Example

Input:
Hello Friend

Output:
வணக்கம் நண்பரே
